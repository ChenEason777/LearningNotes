### 1. 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

### 2. 文字样式

```markdown
**加粗**
*斜体*
~~删除线~~
```

### 3. 列表

* 无序列表

```markdown
- 第一项
- 第二项
```

* 有序列表

```markdown
1. 第一步
2. 第二步
```

### 4. 代码（最常用）

* 行内代码

```markdown
`ls -l` `git push`
```

* 代码块（高亮）

```markdown
```bash
ls -l
cd /home
```


### 5. 链接

```
### 6. 图片

```

### 9. 表格（超适合笔记）
```

| 命令 | 说明 |
| --- | --- |
| ls | 查看文件 |
| cd | 切换目录 |

```
---














GitHub Markdown 里写​**注释**​（别人看不见、只有自己编辑时能看到），用这个语法：
```

<!-- 这里写注释，网页上不会显示 -->

```
### 示例


```

# 运维学习笔记

<!-- 这是注释，仅编辑可见 -->

- Linux 命令

```
### 用途


* 写备忘、待办
* 临时隐藏内容
* 给笔记加备注
```
